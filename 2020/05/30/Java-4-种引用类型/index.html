Java 4 种引用类型

Posted on 2020-05-30 |

强引用

  • JAVA 默认引用模式，不会被java gc 主动回收
    • 可能引起内存泄漏
    • 可能导致 OutOfMemory
  • JAVA 代码直接调用获取实例

软引用

  • 内存不足时，就会回收软引用对象
    • 不会引起内存泄漏
  • 应用场景
    • 内存敏感的高速缓存
    • 图片编辑器
    • 视频编辑器
  • 可以配合引用队列联合使用
  • get 方法获取实例

弱引用

  • java gc的时候就会被回收
    • 不会引起内存泄漏
  • get 方法获取
  • 也可与引用队列配合使用
  • WeakHashMap

虚引用

  • java gc的时候被回收
    • 可能引起内存泄漏
  • 应用场景
    • 跟踪对象被垃圾回收器回收的活动
  • 必须和引用队列绑定
  • 不可获取
